# Inventory

created to keep track of inventory.


## Installation
    git clone repo
    '$ npm i' to get all node_modules

## How To Run
    in 2 different cmds or terminals run these 2 commands:
        1st:
           $ cd into server
           $ run node server
        2nd:
           $ cd into client
           $ run npm start

## Usage
    localhost:3000 on chrome is the best outlook on the it
